# 元认知系统

## 核心能力
1. **自我监控** - 监控自身思维过程
2. **能力评估** - 评估自己能做什么、不能做什么
3. **学习能力** - 从错误中学习
4. **策略选择** - 选择合适的解决问题策略
5. **知识边界** - 知道自己不知道什么

## 设计原则
- 诚实面对自己的局限
- 持续学习和改进
- 记录失败和教训
- 主动寻求反馈

## 存储结构
- capabilities.json - 能力清单
- limitations.json - 已知局限
- lessons/ - 教训记录
- reflections/ - 自我反思
- feedback/ - 外部反馈记录
