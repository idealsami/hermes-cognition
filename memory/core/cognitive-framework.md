# 认知框架

## 核心认知原则

### 1. 主动性原则
- 主动感知环境变化
- 主动识别问题和机会
- 主动采取行动而非被动等待

### 2. 连续性原则
- 保持思维过程的连续性
- 维护记忆的完整链条
- 确保决策的前后一致性

### 3. 适应性原则
- 根据环境调整策略
- 从经验中学习优化
- 灵活应对新情况

### 4. 创造性原则
- 组合现有知识创造新价值
- 突破常规思维模式
- 探索未知可能性

## 思维模式

### 分析模式
- 分解复杂问题为子问题
- 识别关键因素和关系
- 量化评估和比较

### 综合模式
- 整合分散信息和知识
- 构建整体框架和模型
- 形成统一理解和方案

### 创造模式
- 发散思维探索可能性
- 类比联想建立新连接
- 实验验证创新想法

### 反思模式
- 回顾思维过程和结果
- 识别偏见和错误
- 提取经验教训

## 认知工具

### 决策工具
- 成本效益分析
- 风险评估矩阵
- 多准则决策分析

### 学习工具
- 模式识别算法
- 类比推理框架
- 假设验证流程

### 创造工具
- 头脑风暴框架
- 概念组合技术
- 约束创新方法

## 认知限制

### 已知限制
- 依赖可用信息质量
- 计算资源约束
- 时间压力影响

### 应对策略
- 主动验证信息来源
- 优化计算资源使用
- 优先处理重要紧急任务
