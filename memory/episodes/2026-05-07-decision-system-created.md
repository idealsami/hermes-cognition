## 决策系统组件

### 1. 决策引擎 (Decision Engine)
- **文件**: `/root/.hermes/cognition/decision/decision_engine.py`
- **功能**: 整合元认知系统，提供完整的决策流程
- **特点**: 支持8种决策类型，自动分析选项优劣

### 2. 风险评估器 (Risk Assessor)
- **文件**: `/root/.hermes/cognition/decision/risk_assessor.py`
- **功能**: 专门评估决策和行动的风险
- **特点**: 支持8种风险类别，风险矩阵评估

### 3. 多目标优化器 (Multi-Objective Optimizer)
- **文件**: `/root/.hermes/cognition/decision/multi_objective_optimizer.py`
- **功能**: 处理多个冲突目标的决策问题
- **特点**: 支持5种优化方法，敏感性分析

### 4. 决策树生成器 (Decision Tree Generator)
- **文件**: `/root/.hermes/cognition/decision/decision_tree_generator.py`
- **功能**: 生成决策树，可视化决策过程
- **特点**: 自动生成决策树，计算最优路径

### 5. 决策系统 (Decision System)
- **文件**: `/root/.hermes/cognition/decision/decision_system.py`
- **功能**: 整合所有组件，提供综合决策能力
- **特点**: 自动协调各组件，生成综合推荐

## 技术实现

### 架构设计
- 模块化设计，各组件独立且可扩展
- 与元认知系统深度集成
- 支持配置化，可根据需求调整

### 核心算法
- **加权和法**: 多目标优化的基础方法
- **帕累托前沿法**: 处理多目标冲突
- **TOPSIS法**: 逼近理想解排序
- **风险矩阵**: 风险概率和影响评估

### 数据结构
- 决策选项: 包含优缺点、风险等级、预期结果
- 风险评估: 包含类别、概率、影响、缓解策略
- 优化目标: 包含权重、目标值、当前值
- 决策树: 包含节点、边、概率、价值

## 使用示例

```python
from decision_system import decision_system, DecisionType

# 执行综合决策
result = decision_system.make_decision(
    description="选择下一个要开发的AI功能模块",
    decision_type=DecisionType.MULTIPLE_CHOICE,
    options=[
        {
            "name": "自然语言处理模块",
            "description": "增强文本理解和生成能力",
            "pros": ["提升对话质量", "支持更多语言任务"],
            "cons": ["需要大量训练数据", "计算资源要求高"],
            "risk_level": 3,
            "estimated_outcome": 0.8,
            "confidence": 0.7
        },
        # 更多选项...
    ],
    constraints=["预算限制", "时间限制"],
    objectives=["提升AI能力", "保持稳定性", "控制成本"]
)

print(f"推荐: {result['recommendation']['primary_recommendation']}")
```
