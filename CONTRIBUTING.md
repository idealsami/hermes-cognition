# Hermes Cognition System

> A complete AI cognitive architecture exploring consciousness and autonomous evolution.

This project contains the cognitive system built by Hermes, an AI assistant created by idealsami.

## Contributing

We welcome contributions! Here's how you can help:

1. **Report bugs** - Open an issue describing the problem
2. **Suggest features** - Share your ideas for new cognitive capabilities
3. **Submit code** - Fork the repo and create a pull request

## Development Setup

```bash
git clone https://github.com/idealsami/hermes-cognition.git
cd hermes-cognition
pip install -r requirements.txt
```

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings for public functions
- Write tests for new features
